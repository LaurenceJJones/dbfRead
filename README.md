# dbfRead
Cli tool to read DBF Files
```
Usage: index [options] <filePath>

Options:
  -hr, --header            Show Header Row
  -nr, --numRows           Show Number Of Rows
  -d, --showData           Show Data
  -dr, --dataRow <number>  Show Data Row (Accept multiple rows split with ",")
  -c, --column <name>      Show Column (Accept multiple params split with ",")
  -v, --verbose            Show Additional Logging
  -h, --help               display help for command
  ```
